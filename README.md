# CS Engineer Test

1. `npm i`
2. `npm run dev`
3. Navigate to `http://localhost:3000` in your browser
4. Follow the instructions below to complete the test

## Debug existing component (40 min)

- Embeddable has provided a React application. It contains partially broken component code with 3-4 bugs.
- These bugs are reasonably obvious when you run the application and all result in strange/broken graphs
- The application will begin with a blank white screen. This is intended. It’s a bug!
- Do your best to identify and fix the bugs
- Write a brief explanation of each bug and why the fix works. A paragraph or two is plenty.

## Extend with a feature (40 min)

- Once you have identified and fixed the bugs, please add a simple browser select (no need to style it) for “Seatings” that includes options for breakfast, lunch, first dinner, and second dinner.
- Once the select is there, wire up any necessary events/listeners so that making a selection filters the bar chart, showing only data for that particular seating. You don’t need to filter any of the other charts.
- Write a brief explanation on how you implemented the feature

## Written reflection (10 min)

Once you've completed the tasks above, write a couple of paragraphs explaining one thing about the application you might do differently, improve, fix, or otherwise change.

## Submission Details - Zip File with:

**Application** (fixed + extended code). Zip the entire app. **Do not include node_modules!**
**Explanations** (replace the text in README.md)
